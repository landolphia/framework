framework
=========

A Java game loop.

#Quickstart

Compile with
```javac -cp ../jar/disruptor.jar:../jar/lwjgl.jar:../jar/lwjgl_util.jar: -d . *.java```

Run with
```java -classpath ../jar/disruptor.jar:../jar/lwjgl.jar:../jar/lwjgl_util.jar: -Djava.library.path=../native/linux/x64/ framework.GameLoop```
